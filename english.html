Battleship Game in Python

Project developed in Python as part of the Operating Systems course, focused on logic, organization, and flow control. The game simulates a battleship match between the player and the system, using matrices, validations, and modular code structure. The project reinforced concepts of processes, decision-making, and programming best practices.

Technologies and concepts: Python, programming logic, matrices, modularization, flow control.
