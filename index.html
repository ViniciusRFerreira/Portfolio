Jogo de Batalha Naval em Python

Projeto desenvolvido em Python como parte da disciplina de Sistemas Operacionais, com foco em lógica, organização e controle de fluxo. O jogo simula uma batalha naval entre jogador e sistema, utilizando matrizes, validações e estruturação modular do código. O projeto reforçou conceitos de processos, tomada de decisão e boas práticas de programação.

Tecnologias e conceitos: Python, lógica de programação, matrizes, modularização, controle de fluxo.

Projeto de Análise de Dados com Python

Projeto acadêmico voltado à análise de dados, utilizando Python para tratamento, exploração e interpretação de conjuntos de dados. Foram aplicadas técnicas de manipulação, visualização e análise estatística para extração de insights e compreensão de padrões, com foco em organização e clareza dos resultados.

Tecnologias e conceitos: Python, Pandas, NumPy, análise de dados, regressão linear.

Jogo Front-end Publicado na Play Store

Desenvolvimento de um jogo utilizando React Native como framework, com foco em interatividade e experiência do usuário. O projeto foi finalizado, testado e publicado na Play Store, proporcionando experiência prática com o ciclo completo de desenvolvimento, desde a implementação até a entrega final.

Tecnologias e conceitos: React Native, JavaScript/TypeScript, lógica de interface, UX, publicação de aplicações.

Jogo Educacional Infantil - Separação do Lixo

Mini jogo educacional desenvolvido com foco no público infantil, voltado para a conscientização e aprendizado sobre a separação correta do lixo. O projeto teve como objetivo unir tecnologia e impacto social, utilizando uma interface simples, interativa e didática.

Tecnologias e conceitos: Vue.js, JavaScript, componentes, lógica de interação, educação ambiental.

Consulta de Endereço via API - Java (POO)

Aplicação desenvolvida em Java que consome uma API de terceiros para consulta automática de endereços a partir do CEP. O projeto foi estruturado utilizando Programação Orientada a Objetos, com foco em organização, reutilização de código e boas práticas de arquitetura.

Tecnologias e conceitos: Java, Programação Orientada a Objetos, consumo de API REST, estruturação de código.
